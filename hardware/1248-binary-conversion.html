Binary to Decimal

Let's start with some simple examples:

Can you quickly determine the decimal values for these three binary numbers without memorizing them?

For 110: treating 0 as false and 1 as true, the positions with "true" (1) are 4 and 2. Adding them together: 4+2=64 + 2 = 64+2=6. This is the decimal value of 110.

The positions with 1 are 8 and 2. Adding them gives 8+2=108 + 2 = 108+2=10.

This makes decimal to binary conversion simple. But what about larger numbers, like 101010?

The sequence continues: 1, 2, 4, 8, 16, 32, 64, 128... These are powers of 2.

Sum the numbers where the bit is 1: 32+8+2=4232 + 8 + 2 = 4232+8+2=42. The decimal value of 101010 is 42.

Decimal to Binary

Now, let's calculate the binary form for these three numbers:

  • 23
  • 77

Using the 1248 method, we decompose the number 9 into powers of 2: 9=8+19 = 8 + 19=8+1. Mapping this to the table:

The "8" and "1" positions are set to 1, and the others are 0. The result is 1001.

For 23, decompose it: 23=16+7=16+4+2+123 = 16 + 7 = 16 + 4 + 2 + 123=16+7=16+4+2+1.
