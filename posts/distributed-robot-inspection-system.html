еҲҶеёғејҸжңәеҷЁдәәеӨҡжңәеҚҸеҗҢе·ЎжЈҖзі»з»ҹи®ҫи®Ў

иҝҷзҜҮж–Үз« жҳҜеҜ№д№ӢеүҚеҒҡиҝҮзҡ„еҲҶеёғејҸжңәеҷЁдәәеӨҡжңәеҚҸеҗҢзі»з»ҹзҡ„дёҖдәӣи®ҫи®Ўе’ҢжҖқиҖғзҡ„ж•ҙзҗҶеҪ’зәігҖӮеҶҷдёӢжқҘдё»иҰҒжҳҜдёәдәҶеӨҚзӣҳжҖқи·ҜпјҢзҗҶжё…жһ¶жһ„пјҢеҗҢж—¶д№ҹиғҪз»ҷеҗҺжқҘиҖ…жҸҗдҫӣдёҖдәӣеҸӮиҖғгҖӮ

дёҖгҖҒеҹәжң¬жҰӮеҝө

жҲ‘д»¬иҰҒи§ЈеҶізҡ„й—®йўҳжҳҜпјҡеҰӮдҪ•и®©еӨҡдёӘејӮжһ„жңәеҷЁдәәи®ҫеӨҮеҚҸеҗҢе®ҢжҲҗе·ЎжЈҖд»»еҠЎгҖӮиҝҷж„Ҹе‘ізқҖзі»з»ҹйңҖиҰҒз»ҹдёҖи°ғеәҰж— дәәжңәгҖҒең°йқўжңәеҷЁдәәгҖҒеӣәе®ҡзӣёжңәе’Ңе…¶д»–иҫ№зјҳи®Ўз®—иҠӮзӮ№зӯү

д»Һжһ¶жһ„еұӮйқўзңӢпјҢжңҖеҗҲзҗҶзҡ„ж–№ејҸжҳҜ дёӯеҝғжҺ§еҲ¶пјҡ

  • дә‘з«ҜиҙҹиҙЈе…ЁеұҖи°ғеәҰдёҺд»»еҠЎзј–жҺ’пјҢдёӢеҸ‘з»ҹдёҖзҡ„д»»еҠЎжҢҮд»Өпјӣ
  • еҗ„и®ҫеӨҮеҸӘйңҖжҢүз…§еҲҶй…Қзҡ„д»»еҠЎжү§иЎҢпјҢе№¶е°ҶиҝҗиЎҢзҠ¶жҖҒдёҠжҠҘдә‘з«Ҝ

дёәдәҶи®©дёҚеҗҢзұ»еһӢзҡ„и®ҫеӨҮпјҲеҺӮе•ҶдёҚеҗҢгҖҒеҚҸи®®дёҚеҗҢпјүйғҪиғҪиў«и°ғеәҰпјҢйңҖиҰҒдёҖеҘ—з»ҹдёҖзҡ„и§„иҢғгҖӮиҝҷе°ұеј•еҮәдёӨдёӘе…ій”®зӮ№пјҡ

  1. д»»еҠЎ DSLпјҲDomain-Specific LanguageпјҢйўҶеҹҹдё“з”ЁиҜӯиЁҖпјүпјҡж ҮеҮҶеҢ–зҡ„д»»еҠЎжҸҸиҝ°иҜӯиЁҖпјҢз”ЁжқҘиЎЁиҫҫвҖңд»»еҠЎжҳҜд»Җд№ҲпјҢеҰӮдҪ•жү§иЎҢвҖқ
  2. и®ҫеӨҮе»әжЁЎпјҲзү©жЁЎеһӢ Thing Modelпјүпјҡе°Ҷи®ҫеӨҮзҡ„иғҪеҠӣжҠҪиұЎеҢ–пјҢеҪўжҲҗдёҖдёӘз»ҹдёҖзҡ„жҺҘеҸЈеұӮ

иҝҷж ·пјҢдә‘з«Ҝе°ұеҸҜд»ҘеҸӘе…іеҝғвҖңеҒҡд»Җд№ҲвҖқпјҢиҖҢдёҚз”Ёе…іеҝғвҖңи®ҫеӨҮжҳҜжҖҺд№ҲеҒҡзҡ„вҖқ

зү©жЁЎеһӢпјҲThing Modelпјү

иҝҷйҮҢеҖҹз”Ё DJI зҡ„ TSLпјҲThing Specification LanguageпјүжҰӮеҝө
зү©жЁЎеһӢжң¬иҙЁдёҠжҳҜдёҖдёӘ и®ҫеӨҮзҡ„ж•°еӯ—еҢ–иЎЁзӨәпјҢд»ҺдёүдёӘз»ҙеәҰжҸҸиҝ°дәҶи®ҫеӨҮзҡ„иғҪеҠӣпјҡ

  • еұһжҖ§ (Property)пјҡи®ҫеӨҮзҡ„зҠ¶жҖҒеҸӮж•°пјҢдҫӢеҰӮж— дәәжңәзҡ„е§ҝжҖҒгҖҒз”өжұ з”өйҮҸгҖҒзӣёжңәжЁЎејҸзӯүгҖӮ
  • жңҚеҠЎ (Service)пјҡеҸҜиў«еӨ–йғЁи°ғз”Ёзҡ„еҠҹиғҪпјҢдҫӢеҰӮвҖңиө·йЈһвҖқгҖҒвҖңжӢҚз…§вҖқгҖҒвҖңеҜјиҲӘеҲ°жҹҗеқҗж ҮзӮ№вҖқгҖӮ
  • дәӢд»¶ (Event)пјҡи®ҫеӨҮиҝҗиЎҢиҝҮзЁӢдёӯи§ҰеҸ‘зҡ„дәӢд»¶пјҢдҫӢеҰӮвҖңиҝӣе…ҘзҰҒйЈһеҢәвҖқгҖҒвҖңз”өйҮҸдёҚи¶іе‘ҠиӯҰвҖқгҖӮ

дёҖдёӘе…ёеһӢзҡ„зү©жЁЎеһӢ JSON з»“жһ„еӨ§иҮҙеҰӮдёӢпјҡ

{
    "profile": {},    	# иЎЁзӨәиҜҘдә§е“ҒжҳҜд»Җд№Ҳ
    "properties": [],   # иЎЁзӨәиҜҘдә§е“Ғжңүе“ӘдәӣеұһжҖ§
    "services": [],   	# иЎЁзӨәиҜҘдә§е“Ғжңүе“ӘдәӣеҸҜд»Ҙиў«еӨ–йғЁи°ғз”Ёзҡ„иғҪеҠӣе’Ңж–№жі•
    "events": []    	# иЎЁзӨәиҜҘдә§е“Ғжңүе“ӘдәӣйңҖиҰҒиў«еӨ–йғЁж„ҹзҹҘе’ҢеӨ„зҗҶзҡ„йҖҡзҹҘдҝЎжҒҜ
}

зү©жЁЎеһӢеёҰжқҘзҡ„жңҖеӨ§еҘҪеӨ„жҳҜпјҡз»ҹдёҖи®ҫеӨҮиғҪеҠӣзҡ„жҠҪиұЎеұӮ
иҝҷе’Ң Linux зҡ„вҖңдёҮзү©зҡҶж–Үд»¶вҖқе“ІеӯҰеҫҲеғҸ вҖ”вҖ” и®ҫеӨҮеҚғе·®дёҮеҲ«пјҢдҪҶеңЁдә‘з«ҜзңӢжқҘпјҢе®ғд»¬йғҪжҳҜдёҖзұ»вҖңиғҪеҠӣйӣҶеҗҲвҖқ

HMS еҒҘеә·з®ЎзҗҶзі»з»ҹ

йҷӨдәҶи®ҫеӨҮиғҪеҠӣзҡ„е®ҡд№үпјҢзі»з»ҹиҝҳйңҖиҰҒе…іжіЁи®ҫеӨҮеҒҘеә·зҠ¶жҖҒгҖӮдҫӢеҰӮж— дәәжңәжҳҜеҗҰжңүж•…йҡңгҖҒең°йқўжңәеҷЁдәәжҳҜеҗҰжҺүзәҝгҖҒз”өжұ жҳҜеҗҰејӮеёё

еҒҘеә·з®ЎзҗҶпјҲHMS, Health Management Systemпјүзҡ„ж ёеҝғд»·еҖјеңЁдәҺпјҡ

  • жҸҗеүҚйў„иӯҰпјҡеҸ‘зҺ°и®ҫеӨҮжҪңеңЁй—®йўҳпјҢйҳІжӯўд»»еҠЎдёӯж–ӯ
  • и§„еҲҷеҢ–еӨ„зҗҶпјҡйҖҡиҝҮи§„еҲҷеј•ж“Һе®ҡд№үе‘ҠиӯҰдёҺеӨ„зҗҶж–№ејҸпјҲжҜ”еҰӮдҪҺз”өйҮҸиҮӘеҠЁиҝ”иҲӘпјү
  • иҮӘеҠЁеҢ–жҒўеӨҚпјҡжҹҗдәӣжғ…еҶөдёӢи®ҫеӨҮеҸҜд»ҘиҮӘж„ҲпјҢжҲ–иҖ…зі»з»ҹиҮӘеҠЁйҮҚж–°еҲҶй…Қд»»еҠЎ

иҝҷйҮҢеҸҜд»ҘеҸӮиҖғ RuleGoпјҢдёҖдёӘиҪ»йҮҸзҡ„и§„еҲҷеј•ж“Һеә“пјҢйқһеёёйҖӮеҗҲеҒҡи®ҫеӨҮеҒҘеә·з®ЎзҗҶе’Ңд»»еҠЎи§„еҲҷеӨ„зҗҶгҖӮ

AI дёҺиҮӘз„¶иҜӯиЁҖд»»еҠЎзј–жҺ’

еҪ“жҲ‘д»¬жңүдәҶд»»еҠЎ DSL е’Ңзү©жЁЎеһӢеҗҺпјҢдјҡеҸ‘зҺ°иҝҷе’Ң AI зҡ„ MCPпјҲModel Context Protocolпјү жҖқи·Ҝйқһеёёзұ»дјјпјҡ

  • зү©жЁЎеһӢе®ҡд№үи®ҫеӨҮиғҪеҠӣ
  • д»»еҠЎ DSL е®ҡд№үи°ғз”Ёж–№ејҸ

иҝҷе°ұж„Ҹе‘ізқҖпјҡд»»еҠЎе®Ңе…ЁеҸҜд»Ҙз”ЁиҮӘз„¶иҜӯиЁҖжҸҸиҝ°пјҢеҶҚз”ұ AI иҪ¬жҚўжҲҗ DSL
дҫӢеҰӮпјҡ

вҖңи®©ж— дәәжңәйЈһеҲ°зӮ№ A е·ЎжЈҖпјҢеҗҢж—¶и®©ең°йқўжңәеҷЁдәәеҺ»зӮ№ BвҖқ

AI е°ұиғҪзҝ»иҜ‘жҲҗ DSL JSON

- `steps`: д»»еҠЎжӯҘйӘӨж•°з»„пјҢжҜҸдёӘжӯҘйӘӨдёәвјҖдёӘд»»еҠЎеҠЁдҪңпјҢе…·жңүд»ҘдёӢз»“жһ„пјҡ
жҜҸдёӘ step вҪҖжҢҒд»ҘдёӢеӯ—ж®өпјҡ
- `type`: д»»еҠЎеҠЁдҪңзұ»еһӢпјҢдҫӢеҰӮ `drone.goto`, `dog.capture`

- `params`: жү§вҫҸеҠЁдҪңжүҖйңҖеҸӮж•°пјҲж №жҚ®и®ҫеӨҮиғҪвј’пјү
- `retry`: пјҲеҸҜйҖүпјүеӨұиҙҘйҮҚиҜ•ж¬Ўж•°пјҢдҫӢеҰӮ 2
- `onError`: пјҲеҸҜйҖүпјүеӨұиҙҘж—¶зҡ„еӨҮвҪӨеҠЁдҪңпјҢдҫӢеҰӮ `{ "type": "drone.return" }`
- `condition`: пјҲеҸҜйҖүпјүд»…еңЁж»ЎвҫңжқЎд»¶ж—¶жү§вҫҸпјҢдҫӢеҰӮ `"battery > 30"`
- `type: "group"` иЎЁзӨәе№¶еҸ‘жӯҘйӘӨпјҢеҢ…еҗ«вјҖдёӘ `tasks` ж•°з»„

  • и®ҫеӨҮиғҪеҠӣеҢ№й…ҚпјҡйҖҡиҝҮзү©жЁЎеһӢиҮӘеҠЁйҖӮй…ҚпјҢдёҚеҗҢеҺӮе•Ҷзҡ„и®ҫеӨҮйғҪиғҪеҚіжҸ’еҚіз”Ё

ж ёеҝғжңҚеҠЎжЁЎеқ—пјҡ

  • д»»еҠЎи°ғеәҰзј–жҺ’еј•ж“Һ
  • и§„еҲҷеј•ж“ҺпјҲеҒҘеә·з®ЎзҗҶгҖҒејӮеёёеӨ„зҗҶпјү
  • HMS жңҚеҠЎпјҲи®ҫеӨҮзҠ¶жҖҒзӣ‘жҺ§пјү
  • иҫ№зјҳеұӮжңҚеҠЎ edge-sdk

жҜ”еҰӮдёҖдёӘвҖңеҢәеҹҹ A е…Ёйқўе·ЎжЈҖвҖқд»»еҠЎпјҢдјҡиў« Mission Service жӢҶеҲҶжҲҗпјҡ

  1. йҖҡзҹҘжңәе·ўејҖжңәпјҢжЈҖжҹҘж— дәәжңәзҠ¶жҖҒ
  2. ж— дәәжңәиө·йЈһпјҢеүҚеҫҖеқҗж Ү A1-A5 жӢҚз…§дёҠдј 
  3. еҗҢж—¶пјҢең°йқўжңәеҷЁдәәе·ЎжЈҖ A6-A9
  4. жүҖжңүж•°жҚ®жұҮжҖ»еҲ° Telemetry
  5. д»»еҠЎе®ҢжҲҗеҗҺпјҢж— дәәжңәиҝ”иҲӘпјҢжңәеҷЁдәәеӣһеҲқе§ӢзӮ№

иҝҷз§ҚеӨҡжңәеҚҸеҗҢпјҢжң¬иҙЁдёҠжҳҜд»»еҠЎ DSL дёҺи®ҫеӨҮзү©жЁЎеһӢзҡ„з»“еҗҲгҖӮ

DAGпјҲжңүеҗ‘ж— зҺҜеӣҫпјүиЎЁиҫҫд»»еҠЎдҫқиө–е…ізі»DSL д»»еҠЎжӯҘйӘӨз»“жһ„

DAG еңЁиҝҷйҮҢе°Өе…¶йҮҚиҰҒпјҡд»»еҠЎеҫҖеҫҖдёҚжҳҜзәҝжҖ§зҡ„пјҢиҖҢжҳҜдёҖдёӘжңүдҫқиө–зҡ„еӣҫгҖӮйҖҡиҝҮ DAGпјҢжҲ‘д»¬еҸҜд»ҘзІҫзЎ®жҸҸиҝ°вҖңе“ӘдёӘжӯҘйӘӨдҫқиө–дәҺе“ӘдёӘжӯҘйӘӨвҖқпјҢйҒҝе…Қж··д№ұ
