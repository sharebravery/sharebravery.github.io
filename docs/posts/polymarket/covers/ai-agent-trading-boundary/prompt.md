# 封面图设计

## 文章主题
OpenClaw 能交易 Polymarket 吗：AI Agent 在预测市场的边界

## 核心概念
- AI Agent 与交易的边界
- 人机协作的分工
- 执行用代码，AI做筛选，策略来自人

## 视觉方向
- 技术感、简洁
- 分界线/边界的概念
- 对比：机器执行 vs 人类判断

## 配色
- 深色背景（科技感）
- 青色/蓝色（AI、代码）
- 橙色/金色（人类判断、决策）

## 图片 Prompt

A conceptual illustration showing the boundary between AI and human in trading. Split composition: left side shows abstract AI/automation elements (code, neural network patterns, clean geometric shapes) in cool cyan/blue tones; right side shows human decision elements (abstract mind, strategic thinking, charts) in warm orange/gold tones. A clear dividing line in the center represents the boundary. Minimalist tech aesthetic, dark background, professional editorial style. 16:9 aspect ratio.
